How PHP is Used in Database Programming

PHP is one of the most widely used server-side scripting languages for developing web applications, and it plays a crucial role in connecting web pages with databases. One of its main strengths is its ability to interact with databases like MySQL, PostgreSQL, SQLite, and others using built-in functions or libraries.

With PHP, developers can perform tasks such as:

  • Connecting to a database
  • Inserting, updating, and deleting records
  • Retrieving data from tables
  • Displaying dynamic content based on database queries
  • Implementing login systems, user profiles, and content management

Example: Connecting to MySQL and Fetching Data


<?php
$host = "localhost";
$username = "root";
$password = "";
$dbname = "my_database";

// Create connection
$conn = new mysqli($host, $username, $password, $dbname);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

// Select data
$sql = "SELECT id, name FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    // Output each row
    while($row = $result->fetch_assoc()) {
        echo "ID: " . $row["id"]. " - Name: " . $row["name"]. "<br>";
    }
} else {
    echo "0 results";
}

$conn->close();
?>

This code connects to a MySQL database, retrieves user data, and displays it on a webpage. This demonstrates how PHP can interact with databases to build dynamic websites and applications.
