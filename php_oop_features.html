PHP Object-Oriented Programming (OOP) Features

1. Classes and Objects

In PHP, a class is a blueprint for creating objects. An object is an instance of a class.


class Car {
    public $color;
    public $brand;

    function __construct($color, $brand) {
        $this->color = $color;
        $this->brand = $brand;
    }

    function describe() {
        return "A $this->color $this->brand car.";
    }
}

$myCar = new Car("red", "Toyota");
echo $myCar->describe();  // Output: A red Toyota car.

2. Inheritance

Inheritance allows one class (child) to inherit the properties and methods of another class (parent).


class Animal {
    public $name;

    function __construct($name) {
        $this->name = $name;
    }

    function speak() {
        return "Animal sound";
    }
}

class Dog extends Animal {
    function speak() {
        return "$this->name says Woof!";
    }
}

$dog = new Dog("Buddy");
echo $dog->speak();  // Output: Buddy says Woof!

3. Encapsulation

Encapsulation restricts access to certain details of an object and exposes only necessary components.


class Person {
    private $name;

    function __construct($name) {
        $this->name = $name;
    }

    public function getName() {
        return $this->name;
    }
}

$person = new Person("John");
echo $person->getName();  // Output: John

4. Polymorphism

Polymorphism allows different objects to be used interchangeably if they implement the same method signature.


class Cat extends Animal {
    function speak() {
        return "$this->name says Meow!";
    }
}

$cat = new Cat("Misty");
echo $cat->speak();  // Output: Misty says Meow!

5. Abstraction

Abstraction defines a structure without implementing specifics. Child classes are expected to implement the details.


abstract class Shape {
    abstract public function area();
}

class Circle extends Shape {
    public $radius;

    function __construct($radius) {
        $this->radius = $radius;
    }

    public function area() {
        return pi() * $this->radius * $this->radius;
    }
}

$circle = new Circle(5);
echo $circle->area();  // Output: 78.5398
